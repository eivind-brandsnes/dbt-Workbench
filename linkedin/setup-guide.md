# LinkedIn Page Setup Guide for dbt-Workbench

## Step-by-Step Instructions

### 1. Create the Company Page

1. Go to [LinkedIn Create Company Page](https://www.linkedin.com/company/setup/new/)
2. Sign in with your personal LinkedIn account
3. Fill in the following details:

| Field | Value |
|-------|-------|
| **Company Name** | dbt-Workbench |
| **Company Logo** | Use `assets/brand.svg` converted to PNG (300x300px) |
| **Tagline** | Open-source UI for dbt â€” model browsing, lineage visualization, run orchestration |
| **Industry** | Software Development |
| **Company Size** | 1-10 employees (or "Open Source" if available) |
| **Company Type** | Privately Held |
| **Headquarters** | Your location |
| **Website** | https://github.com/dbt-workbench/dbt-Workbench |
| **LinkedIn URL** | `dbt-workbench` (custom URL once verified) |

### 2. Complete the About Section

Copy the content from `linkedin/content/about-section.md`

### 3. Upload Banner

- Use banner from `linkedin/assets/banner-specs.md` specifications
- Size: 1128x191 px minimum, 1128x191 px recommended
- Show UI screenshot with key features annotated

### 4. Add Page Details

#### Products & Services
- Product 1: **Model Browser** - Browse dbt models, sources, and tests with detailed metadata
- Product 2: **Lineage Visualization** - Interactive model and column-level lineage graphs
- Product 3: **Run Orchestration** - Execute dbt commands with real-time log streaming
- Product 4: **Scheduler** - Cron-style scheduled runs with notifications
- Product 5: **SQL Workspace** - Query database with syntax highlighting and model support
- Product 6: **AI Copilot** - AI-powered assistance for SQL and dbt queries

#### Featured Hashtags
```
#dbt #DataEngineering #OpenSource #DataAnalytics #AnalyticsEngineering
#DataLineage #ETL #DataPipeline #DuckDB #FastAPI #React
```

### 5. Set Up Administrators

1. Go to Page Admin settings
2. Add 2-3 core team members as:
   - Super Admin (1 person) - Full access
   - Content Admin (1-2 people) - Can post and manage content
3. Configure notification preferences:
   - Daily digest for all admins
   - Immediate notifications for page mentions

### 6. Enable Analytics

1. Go to Page Analytics
2. Review initial metrics
3. Set up:
   - Weekly performance reports
   - Monthly comprehensive review
4. Key metrics to track:
   - Follower growth rate
   - Engagement rate (likes + comments + shares / impressions)
   - Post reach
   - Visitor demographics

### 7. Follow Related Pages

Build your network by following:
- **Competitors/Similar Projects:** dbt Labs, Airbyte, Dagster, Prefect
- **Companies in Ecosystem:** Snowflake, Databricks, dbt Labs, Apache Airflow
- **Open Source Projects:** Apache Spark, Pandas, SQLAlchemy
