## ðŸ†˜ Need Help?

### Questions?
- **Setup:** Refer to `setup-guide.md`
- **Content:** Use templates in `content/post-templates.md`
- **Design:** See `assets/banner-specs.md`
- **Advocacy:** Review `content/employee-advocacy-guide.md`

### Common Issues

**Banner not uploading?**
- Check file size (must be under 5MB)
- Verify dimensions (1128x191px minimum)
- Try PNG format instead of JPG

**Low engagement?**
- Post at optimal times (Tue-Thu, 8-10 AM)
- Use questions to encourage comments
- Share authentic stories vs. marketing
- Experiment with different formats (text, image, video)

**Can't find good visuals?**
- Take clean UI screenshots
- Create simple diagrams in Figma
- Use code snippets in monospace font
- Keep it minimal and clean
