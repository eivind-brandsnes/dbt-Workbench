# dbt docs viewer

The **dbt docs viewer** in dbt-Workbench serves the `dbt docs generate` output directly from mounted artifacts. You get the full dbt documentation experience inside the same UI used for lineage and run orchestration.

## Enable docs previews

1. Run `dbt docs generate` in your dbt project.
2. Mount the generated `target/` directory into the `DBT_ARTIFACTS_PATH` location.
3. Ensure the directory contains `index.html` plus assets.

The UI automatically serves the latest docs assets with the rest of your artifacts.
