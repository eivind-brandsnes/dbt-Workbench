# How to run dbt from a web UI

This guide walks through orchestrating dbt runs directly in dbt-Workbench.

## Steps

1. Configure an environment with the correct dbt profile/target.
2. Open **Run Orchestration** in the UI.
3. Click the execution button for the desired command.
4. Monitor streaming logs and review artifacts when complete.
