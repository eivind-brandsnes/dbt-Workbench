# How to debug failed dbt runs using log streaming

Follow these steps to isolate failures quickly with dbt-Workbench log streaming.

## Steps

1. Open **Run Orchestration** and select a failed run.
2. Review live or stored log streaming output.
3. Inspect run artifacts for additional context.
4. Fix the issue and re-run the command.
