# How to use compiled SQL for safe warehouse querying

Use compiled SQL to execute dbt model logic safely.

## Steps

1. Open a dbt model inside the SQL workspace.
2. Review the compiled SQL pane and confirm the query.
3. Check guardrails for destructive queries.
4. Execute and review profiling results.
