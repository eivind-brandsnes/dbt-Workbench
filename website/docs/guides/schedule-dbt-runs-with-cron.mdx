# How to schedule dbt runs with cron

Use cron-style schedules to automate dbt runs in dbt-Workbench.

## Steps

1. Enable the scheduler and set a default timezone.
2. Create or verify the environment used for runs.
3. Add a schedule with your cron expression.
4. Monitor diagnostics and run history.
