# How to enable JWT auth for a self-hosted dbt UI

Enable JWT authentication for production-grade access control.

## Steps

1. Set `AUTH_ENABLED=true` and configure `JWT_SECRET_KEY`.
2. Apply password policy variables for compliance.
3. Assign roles to users (Viewer, Developer, Admin).
4. Validate access for run orchestration and admin features.
