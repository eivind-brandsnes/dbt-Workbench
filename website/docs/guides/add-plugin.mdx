# How to add a plugin to dbt-Workbench

Extend dbt-Workbench by installing and enabling plugins.

## Steps

1. Review the plugin system specification.
2. Place the plugin directory under `PLUGINS_DIRECTORY`.
3. Enable the plugin from the UI or API.
4. Validate permissions and workspace configuration.
